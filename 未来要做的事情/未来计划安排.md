在[讨论区](https://github.com/slab/quill/discussions/4325)中用如下答案回答用户

使用quilljs-markdown实现quill写markdown实时看见效果需求时，将quill编辑器内容保存到数据库时通过getContents得到delta内容，使用将quill-delta-to-markdown-new将delta转为markdown，数据库就可以保存markdown了；将数据库的markdown内容回显至quill编辑器时，使用markdown-to-quill-delta将markdown转为delta，之后通过setContents设置到编辑器中。
代码如下：
When implementing the requirement of using quilljs-markdown to see the effect of writing markdown in real-time with Quill, the content of the Quill editor is saved to the database by using getContents to obtain the delta content. Then, quill-delta-to-markdown-new(npm i quill-delta-to-markdown-new) is used to convert the delta to markdown, allowing the database to save the markdown.

The code is as follows:

```vue
<template>
  <div class="container">
    <div id="editor"></div>
    <button @click="getMarkdown">Get Markdown</button> |
    <button @click="setMarkdown">Set Markdown</button>
  </div>
</template>

<script setup lang="ts">
import Quill from "quill";
import "quill/dist/quill.core.css";
import "quill/dist/quill.snow.css";
import QuillMarkdown from "quilljs-markdown";
import { deltaToMarkdown } from "quill-delta-to-markdown-new";
import markdownToDelta from "markdown-to-quill-delta";
import { onMounted } from "vue";

let editor: Quill;
const getMarkdown = () => {
  const delta = editor.getContents();
  const markdown = deltaToMarkdown(delta.ops);
  console.log(markdown);
};
const setMarkdown = () => {
  const text =
    "# Hello World\n```\nconsole.log('This is a code block');\n```\n- Item 1\n- Item 2\n";
  const delta = markdownToDelta(text);
  editor.setContents(delta);
};

onMounted(() => {
  editor = new Quill("#editor", {
    debug: "info",
    modules: {
      toolbar: true,
    },
    placeholder: "Compose an epic...",
    theme: "snow",
  });
  const markdownOptions = {
    // You can customize markdown options here
  };
  const quillMarkdown = new QuillMarkdown(editor, markdownOptions);
});
</script>

<style scoped>
.container {
  max-width: 800px;
  margin: 0 auto;
  padding: 20px;
}
</style>

```
