# postgresql

## 安装配置

brew安装postgresql

```bash
brew install postgresql
```

启动postgresql并设置开机自启

```bash
brew services start postgresq
```

brew安装postgresql后，默认基于当前mac用户名使用peer认证，没有密码，终端通过如下方式登录

```bash
psql postgres
```

登录成功后，在psql命令行界面中，为默认用户设置密码

```bash
\password
```

之后在navicat等可视化工具中使用当前mac用户名、刚刚设置的密码进行连接了

## prisma

在Prisma Shema中声明式定义模型，运行命令prisma generate生成Prisma Client，使用Client上的API访问数据库进行操作
